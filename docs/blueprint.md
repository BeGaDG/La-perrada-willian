# **App Name**: Perrada Segura

## Core Features:

- Validación de Administrador: Función de seguridad que verifica si el usuario está autenticado como administrador.
- Menú Público: Permite a los clientes ver la lista de productos (el menú) sin autenticación.
- Gestión de Productos: Permite a los administradores crear, actualizar y eliminar productos del menú.
- Creación de Pedidos: Permite a los clientes crear nuevos pedidos, inicializando el estado como 'PENDIENTE_PAGO'.
- Lectura de Pedidos Propios: Permite a los clientes leer solo los pedidos que han creado (coincidencia de UID).
- Actualización de Pedidos por Administrador: Permite a los administradores actualizar el estado de los pedidos (confirmar pago, enviar a cocina).
- Bloqueo de Eliminación de Pedidos: Impide la eliminación de pedidos para mantener un registro histórico de ventas.

## Style Guidelines:

- Color primario: Rojo ladrillo (#C21D03) para reflejar la identidad de 'La Perrada de William'.
- Color de fondo: Gris claro (#FBFBFB) para proporcionar un contraste suave y facilitar la lectura.
- Color acento: Gris oscuro (#393939) para destacar elementos interactivos y proporcionar una apariencia moderna.
- Fuente principal para cuerpo y encabezados: 'PT Sans', una sans-serif de estilo humanista que combina un aspecto moderno con un toque de personalidad.
- Iconos personalizados que representen cada categoría de producto (perros calientes, hamburguesas, bebidas, etc.).
- Diseño claro y ordenado, con énfasis en la usabilidad y la accesibilidad. Utilizar espaciado consistente para mejorar la legibilidad.
- Animaciones sutiles al cargar nuevos productos o al confirmar un pedido, para proporcionar retroalimentación visual sin distraer al usuario.
